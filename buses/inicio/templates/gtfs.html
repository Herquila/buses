GTFS

La especificación general de suministro de datos de transporte GTFS (General Transit Feed Specification) es un estándar de facto a nivel global para la divulgación de datos abiertos de servicio de transporte público.


Archivo comprimido .zip de documentos .txt con formato de datos .csv.

GTFS permite la visualización de información en aplicaciones como Google Maps y Moovit, entre otras.

GTFS estático

GTFS Static tiene la estructura de una base de datos relacional con información sobre la agencia de transportes, sus rutas, paradas y horarios, entre otros datos que aquí se muestran.

GTFS en tiempo real

GTFS Realtime tiene la estructura de una base de datos relacional con información sobre la agencia de transportes, sus rutas, paradas y horarios, entre otros datos que aquí se muestran.
