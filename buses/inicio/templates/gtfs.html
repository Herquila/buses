GTFS

La especificación general de suministro de datos de transporte GTFS (General Transit Feed Specification) es un estándar de facto a nivel global para la divulgación de datos abiertos de servicio de transporte público.

GTFS permite la visualización de información en aplicaciones como Google Maps y Moovit, entre otras.

GTFS estático

GTFS Static tiene la estructura de una base de datos relacional con información sobre la agencia de transportes, sus rutas, paradas y horarios, entre muchos otros datos.

GTFS en tiempo real

Brinda información adicional al usuario como alertas de estado y actualizaciones de viajes y constituye una opción complementaria a la configuración GTFS estándar.

Puede ser implementado en distintos tipos de aplicaciones tales como planificadores de viajes, sistemas de datos predictivos o herramientas de gestión e investigación. Su flujo de datos es mostrado a continuación (fuente: Banco Mundial):
