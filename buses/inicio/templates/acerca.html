GTFS

La especificación general de suministro de datos de transporte GTFS (General Transit Feed Specification) es un estándar de facto a nivel global para la divulgación abierta de datos de servicio de transporte público.


Archivo comprimido .zip de documentos .txt con formato de datos .csv.

También disponible en OpenMobilityData.

GTFS Estático

GTFS Static tiene la estructura de una base de datos relacional con información sobre la agencia de transportes, sus rutas, paradas y horarios, entre otros datos que aquí se muestran.
